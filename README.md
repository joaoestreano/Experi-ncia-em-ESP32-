# 📡 Meu Aprendizado com ESP32

## 🚀 Sobre este repositório
Este repositório foi criado para documentar minha jornada de aprendizado com o **ESP32**, explorando conceitos de sistemas embarcados, eletrônica e programação.

Aqui você encontrará projetos práticos, testes, experimentos e anotações que mostram minha evolução ao longo do tempo.

---

## 🎯 Objetivos

- Aprender o funcionamento do ESP32 na prática  
- Desenvolver projetos utilizando GPIO, sensores e atuadores  
- Entender comunicação entre dispositivos (Wi-Fi, Bluetooth, etc.)  
- Melhorar minhas habilidades em programação embarcada  
- Criar soluções reais com hardware  

---

## 🧠 Conteúdos abordados

- 🔌 Controle de LEDs (digitalWrite, pinMode)  
- ⏱️ Temporização com `delay()` e `millis()`  
- 🔘 Leitura de botões (input digital)  
- 📡 Conectividade Wi-Fi  
- 📊 Sensores (temperatura, presença, etc.)  
- ⚙️ Integração com outros sistemas  

---

## 🛠️ Tecnologias e Ferramentas

- ESP32  
- Arduino IDE  
- C/C++ (Arduino framework)  
- Protoboard e componentes eletrônicos  

---
